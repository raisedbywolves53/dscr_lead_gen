# Output Specification — Final Excel Workbook

## File

`pipeline/output/leads_YYYY-MM-DD.xlsx`

Generated by `pipeline/scripts/06_score_and_output.py`

---

## Tab Structure

### Tab 1: "All Leads"
Master list of all leads sorted by score (descending). Contains every lead from all ICP segments.

### Tabs 2-N: Per-ICP Segment Tabs
One tab per unique `icp_primary` value. Same columns as All Leads but filtered. Expected tabs:
- Serial Investor (10+)
- STR Operator
- Foreign National
- Entity-Based Investor
- Individual Investor (2-9)
- Multi-Family Investor
- Out-of-State Investor
- Single Investment Property
- Fund Manager / Syndicator

### Final Tab: "Summary"
Dashboard statistics:
- Total leads count
- Breakdown by ICP segment
- Breakdown by tier (1/2/3)
- Contact coverage (% with phone, email, either)
- Investor profile (% entity-owned, STR licensed, foreign, out-of-state)
- Score distribution (mean, median)

## Column Definitions

### Identification

| Column | Type | Description | Source |
|--------|------|-------------|--------|
| lead_id | string | Unique identifier (DSCR-000001 format) | Generated in Step 7 |
| owner_name | string | Property owner name (person or entity) | FDOR NAL OWN_NAME |
| owner_type | string | Individual / LLC / Trust / Partnership / Foreign | Derived from owner_name |
| resolved_person | string | Human name behind entity (if LLC/Corp/Trust) | SunBiz (Step 3) |

### Classification

| Column | Type | Description | Values |
|--------|------|-------------|--------|
| icp_primary | string | Primary ICP segment | See ICP_DEFINITIONS.md |
| icp_secondary | string | Secondary tag (overlay) | STR Operator, Cash-Out Refi Candidate, BRRRR Exit Candidate, Equity Harvest Candidate, Rate Refi Candidate, Growing Portfolio, etc. |
| tier | int | Priority tier | 1 (highest), 2, 3 (lowest) |
| score | int | Lead quality score | 0-100 |

### Contact Information

| Column | Type | Description | Source |
|--------|------|-------------|--------|

| email | string | Email address (if found) | DBPR, Apollo, Hunter |

### Property Portfolio

| Column | Type | Description | Source |
|--------|------|-------------|--------|
| property_count | int | Number of investment properties owned | FDOR NAL (aggregated) |

| most_recent_price | float | Price of most recent purchase | FDOR NAL SALE_PRC1 |

### Refinance Signals

| Column | Type | Description | Source |
|--------|------|-------------|--------|
| estimated_equity | float | Estimated equity per avg property (JV - sale price) | Calculated in Step 2 |

| rate_refi_candidate | bool | 2022-2023 vintage (high-rate financing) | Step 2 |
| equity_harvest_candidate | bool | 30%+ equity available to extract | Step 2 |

## Score Breakdown

The score (0-100) is composed of:

| Factor | Points | Criteria |
|--------|--------|----------|
| Property Count | 0-25 | 1=5, 2-4=10, 5-9=15, 10-19=20, 20+=25 |
| Purchase Recency | 0-20 | <6mo=20, 6-12mo=15, 1-2yr=10, 2-3yr=5 |
| Portfolio Value | 0-15 | <200K=3, 200-500K=6, 500K-1M=9, 1-3M=12, 3M+=15 |
| Entity Sophistication | 0-10 | Entity=5, Multi-entity=10 |
| STR Licensed | 0-10 | Licensed=10 |
| Geographic Fit | 0-10 | South FL=10, Major metro=7, Secondary=5, Rural=3 |
| Contact Available | 0-10 | Phone+email=10, Phone=7, Email=5, Address only=2 |
| **Refi Boost** | **0-40** | **From Step 2 refi_score_boost** |
| **Total** | **0-100** | **Capped at 100** |

## Interpreting the Output

### High-Value Leads (Score 70+)
- Multi-property investors with contact info and refi signals
- STR operators with DBPR license
- Serial investors (10+) in South FL
- Entity-based investors with resolved person + equity

### Medium Leads (Score 50-69)
- 2-5 property investors without full contact info
- Out-of-state investors with moderate portfolios
- Entity owners with some equity signals

### Lower Leads (Score <50)
- Single property investors
- No contact info available
- Rural/secondary market locations
- No refi signals

### Prioritization Recommendations
1. **First**: Score 70+ with phone number → direct call
2. **Second**: Score 50-69 with email → email campaign
3. **Third**: Score 50+ with refi_priority = "High" → refi-focused outreach
4. **Fourth**: STR Operators (any score) → STR-specific DSCR pitch
5. **Fifth**: Foreign Nationals (any score) → foreign national DSCR program
